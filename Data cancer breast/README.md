Breast Cancer Classification using KNN 
 
## Project Overview 
 
This project uses the Breast Cancer Wisconsin Dataset to classify tumors as malignant or benign using the K-Nearest Neighbors (KNN) algorithm. 
 
## Objective 
 
The goal is to develop a machine learning model capable of predicting whether a tumor is malignant or benign based on medical measurements. 
 
## Dataset 
 
Source: 
- Breast Cancer Wisconsin Dataset 
- 569 observations 
- 30 numerical features 
 
## Technologies Used 
 
- Python 
- Pandas 
- NumPy 
- Matplotlib 
- Scikit-Learn 
- Jupyter Notebook 
 
## Methodology 
 
1. Data Cleaning 
2. Exploratory Data Analysis 
3. Feature Scaling using MinMaxScaler 
4. Model Training with KNN 
5. Hyperparameter Testing 
6. Model Evaluation 
 
## Results 
 
Best model: 
 
- Accuracy: 96% 
- Recall (Malignant): 90%+ 
- Precision: 95%+ 
 
The model showed strong performance in identifying malignant tumors while maintaining high overall accuracy. 
 
## Key Learnings 
 
- Importance of feature scaling for KNN 
- Impact of different K values 
- Evaluation using confusion matrix and classification metrics
